Manga API Documentation

Get Manga Information

GET /manga

Retrieve information about a manga and its chapters.

Query Parameters

Parameter Type Description
name string The name of the manga (required)
chapters string Chapter or range of chapters (e.g., "1-3" or "4.5") (required)
source string Source of the manga (default: "weebverse")
quality string Image quality (default: "high")
language string Language of the manga (default: "en")
